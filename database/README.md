# Database

AssetFlow uses PostgreSQL (Neon serverless) managed through Prisma.

- Schema: [`backend/prisma/schema.prisma`](../backend/prisma/schema.prisma)
- Seed: [`backend/prisma/seed.ts`](../backend/prisma/seed.ts)
- Migrations: generated by Prisma under `backend/prisma/migrations/`

The connection string is provided via the `DATABASE_URL` environment variable
in `backend/.env`. The schema itself is defined in a later prompt.
